<h2 align="center">What is PHPPayroll?</h2>

<p align="center">
  <strong>PHPPayroll</strong> is a modern, secure, and flexible
  <strong>Payroll & Attendance Management System</strong> built using
  <strong>PHP</strong> and <strong>MySQL</strong>.
</p>

<p align="center">
  It automates salary calculation, employee attendance, leave management,
  and payroll processing with accuracy and ease.
</p>

<p align="center">
  PHPPayroll supports multiple employment and payment types:
  <br>
  Daily â€¢ Weekly â€¢ Task-Based â€¢ Monthly â€¢ Full-Time â€¢ Part-Time
</p>

<p align="center">
